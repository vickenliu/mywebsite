The differences of map(), filter(), reduce()

map() is an array function to process the every single data stored in the array with the provided specific method passed to the map function. and the every individul data will be passed to the specific method,the method return a processed data to a new array.
filter() is an array function as well, similar with map(), there will be a method passed to the filter() function, the individul data in the array passed to the method, usually the method return a boolean value, only the data proccessed by the method return true will be kept in the array.
reduce() is array function too, The reduce() method applies a function/method against an accumulator and each value of the array (from left-to-right) to reduce it to a single value. the final result of reduce() is return one value result. to achieve this, there are 2 parameters for reduce(), one is method, one is optional initialvalue. every data processed by the method return a value will be the next initialvalue or prev value passed to the method. for the method, we pass in initial value and current value which is the data in the array. the method will excused agaist each value of the array and the final excusion on the last data of the array will return the result for reduce() function.
